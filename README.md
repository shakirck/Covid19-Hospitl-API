# Hospital API  :hospital:  :ambulance:


### Installation


```sh
$ git clone https://github.com/shakirck/Covid19-Hospitl-API.git
$ cd Covid19-Hospitl-API 
$ npm install 
$ npm run dev 
      **OR**
$ npm start 
```
Change the  :key: **Secret Key** in  **key.js** file inside the config folder(***Recommended***)
